Detailed Description

A package for the downloading of Google Maps Static API Images, containing functions that generate API URL's, downloading single images, dual roadmap and satellite images and the Mass downloading of bulk images over a large area.

Returns

  lon,

  subdirectory = 'GMAPS',

  width = 10 
)

Downloads a mass amount of images using a spiral arm extension method.

Using a centre point and a maximum width, it will continue to download images until the max width is reached.

Parameters
lat(float): Starting Latitude (degree) value of Geo-location
lon(float): Starting Longitude (degree) value of Geo-location
user(string): String with Google Static Maps API Key
style(array): Takes a string array of Google-Maps Static API features in groups of 3 as in below Example, [feature1, element1, style1,...] eg ['all', 'labels', 'visibility:off'], default = None
subdirectory(string): Subdirectory to save in, default = "GMAPS"
zoom(int): Zoom level of Google Static Maps API, default = 17
width(float): Max Latitude change (km), default = 10
Returns
NONE: Initiates a Mass Dual Maps download with Given (lat,lon) Co-Ordinates, continuing in spiral pattern until width variable has been reached
